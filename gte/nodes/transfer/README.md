# Transfer

## Splitter

```
         +------------+
         |            | -> outlet_1
inlet -> |  Splitter  | -> outlet_2
         |            | -> outlet_n
         +------------+
```

## Joiner

```
           +----------+
inlet_1 -> |          |
inlet_2 -> |  Joiner  | -> outlet
inlet_n -> |          |
           +----------+
```
